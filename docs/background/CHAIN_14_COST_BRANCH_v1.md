## 2. 成本支线定位
成本支线用于承接售后、专项服务、报价、平台、SIM、仓库等业务中发生的成本记录与成本归集。

当前冻结口径为：
1. 成本支线既服务于保内纯支出，也服务于保外有收入业务
2. 成本支线用于看业务消耗、服务成本和后续毛利
3. 成本支线既承接报价阶段的预估成本，也承接业务执行后的实际成本
4. 间接成本单独管理，当前阶段不展开

---

## 3. 成本的两大口径

### 3.1 保内纯支出
当前冻结口径为：

1. 保内业务可能只有成本支出，没有对应收入
2. 这类成本主要用于反映售后服务消耗与维保负担

### 3.2 保外有收入
当前冻结口径为：

1. 保外业务存在对应收入
2. 这类业务需要同时看收入、成本与毛利
3. 成本可自然关联到报价单、工单或其他业务对象

---

## 4. 报价阶段的预估成本口径
当前冻结口径为：

1. 成本支线除了承接实际成本归集外，也承接报价阶段的预估成本口径
2. 需求单发起时，需要同步带入预估成本信息
3. 若某收费项需要预估服务执行成本，则由服务经理填写预估差旅费与住宿费
4. 其他成本项若系统内已有来源，则在财务第一次制作报价单时可直接带出
5. 财务第一次确认报价时，应基于整张报价单的预估成本判断是否亏钱

说明：
- 该规则的主流程写在报价系统链
- 成本支线只承接预估成本口径与成本来源，不展开报价流程本身

---

## 5. 当前已确认的成本项
当前已确认的正式成本项包括：

1. 服务执行成本
2. 零部件成本
3. 运输成本
4. 平台成本
5. SIM 卡成本
6. 报废损失
7. 办事处 / 分仓房租

说明：
- 间接成本单独管理
- 当前阶段不展开其全部细项

---

## 6. 服务执行成本

### 6.1 服务执行成本分类
当前冻结口径为：

服务执行成本分为两类：

1. 自有人员执行成本
2. 外部服务商执行成本

说明：
- 这里不使用“人工成本”表述
- 因为当前口径下，主要计算的不是工资，而是服务执行过程中发生的差旅、住宿及服务商结算费用

### 6.2 自有人员执行成本
当前冻结口径为：

1. 当服务经理跨驻地城市执行工单时，工单服务执行成本来源于报销支线中归属于工单的实际费用
2. 当前已知可进入工单服务执行成本的自有人员执行费用包括：
   - 差旅费
   - 住宿费
   - 打车费
3. 成本数据来源于出差申请 / 借款 / 补助 / 报销支线的实际报销结果
4. 若一次出差同时处理多个工单，则该次出差对应的服务执行成本在多个工单之间均摊
5. 均摊按当前业务规则直接均摊处理

说明：
- 当前阶段不再只把差旅费和住宿费理解为工单自有人员执行成本来源
- 打车费若已提前单独申请，且对象明确归属于工单，也应进入工单服务执行成本

### 6.3 驻地城市内处理工单
当前冻结口径为：

1. 若服务经理在驻地城市内处理工单
2. 且没有出差报销费用
3. 则该工单服务执行成本记为 0

### 6.4 驻外固定补助
当前冻结口径为：

1. 服务经理驻外时每日存在固定金额补助
2. 该补助不计入单个工单服务执行成本
3. 该补助归入间接成本
4. 原因是该补助还涉及请假与补助计算，当前不放入工单直接成本

### 6.5 外部服务商执行成本
当前冻结口径为：

1. 若工单由外部服务商处理，则该次服务商费用计入工单服务执行成本
2. 服务经理可在服务商上门前填写服务商事前报价
3. 该事前报价属于执行前的成本依据 / 参考，不锁死最终实际成本
4. 外部服务商执行成本只针对已结束工单成立
5. 若服务商最终未实际执行，则该笔外部服务商执行成本不成立
6. 外部服务商按本次服务开具发票
7. 一张发票只对应一张工单
8. 发票金额不要求必须与事前报价一致
9. 工单最终计入成本的外部服务商执行成本，以发票金额为准

说明：
- 外部服务商执行成本不是月结型成本
- 当前按“一张工单一结”理解
- 事前报价用于执行前参考与成本预估
- 最终实际成本不按事前报价锁死，而按服务商发票落入成本

### 6.6 无工单拜访客户的服务执行成本
当前冻结口径为：

1. 存在无工单情况下拜访客户的出差场景
2. 该类出差会产生实际费用
3. 当前已知可进入客户拜访成本的费用包括：
   - 差旅费
   - 住宿费
   - 打车费
4. 该类成本不归到工单，而归到客户拜访成本
5. 成本数据来源于出差申请 / 借款 / 补助 / 报销支线的实际报销结果
6. 若一次出差同时拜访多个客户，则该次客户拜访成本在多个客户之间均摊
7. 均摊按当前业务规则直接均摊处理

说明：
- 出差申请与报销的详细发起、审批、报销流程，不在本文展开
- 本文只冻结该支线流入成本的归集口径
- 当前阶段不再只把差旅费和住宿费理解为客户拜访成本来源
- 打车费若已提前单独申请，且对象明确归属于客户拜访，也应进入客户拜访成本

## 7. 零部件成本
当前冻结口径为：

1. 零部件和消耗品统一纳入零部件成本管理
2. 零部件成本来源于零部件清单中的成本口径
3. 零部件成本可对应到工单
4. 零部件成本也可对应到报价单

说明：
- 当前阶段不单独拆出“消耗品成本”
- 消耗品统一并入零部件成本

---

## 8. 运输成本

### 8.1 零部件运输成本
当前冻结口径为：

零部件运输成本包括：
1. 零部件发给客户的快递费 / 运费
2. 零部件发给分仓的快递费 / 运费
3. 分仓之间调拨产生的运输费
4. 返回产生的运输费

### 8.2 客户委托运输成本
当前冻结口径为：

1. 与物流公司相关的客户委托运输中
2. 我司付给物流公司的费用
3. 计入运输成本
4. 该成本可关联到相应货号和报价单

---

## 9. 平台成本
当前冻结口径为：

1. 平台成本是我司付给平台开发公司的费用
2. 不同平台公司的点数单价不同
3. 平台成本按月上传使用点数
4. 平台成本自然对应到机构代码 / 简称
5. 平台成本与平台费收入是不同口径

---

## 10. SIM 卡成本
当前冻结口径为：

1. SIM 卡成本按月上传账单
2. 账单中记录每张 SIM 卡用了多少钱
3. SIM 卡成本自然对应到 ICCID / 卡号

---

## 11. 报废损失
当前冻结口径为：

1. 总仓报废损失计入成本
2. 分仓报废损失计入成本
3. 报废损失按零部件成本口径处理

---

## 12. 间接成本
当前冻结口径为：

1. 办事处 / 分仓房租属于正式成本口径
2. 驻外固定补助属于间接成本
3. 间接成本单独管理
4. 当前阶段不展开间接成本的完整明细与规则

---

## 13. 成本与业务对象的关系
当前冻结口径为：

1. 服务执行成本（实际）自然关联到工单
2. 服务执行成本（预估）可在报价阶段关联到报价单
3. 无工单拜访客户类出差形成客户拜访成本
4. 零部件成本可关联到工单或报价单
5. 运输成本可关联到工单、货号、报价单或仓库调拨
6. 平台成本自然对应到机构代码 / 简称
7. SIM 卡成本自然对应到 ICCID / 卡号
8. 报废损失自然对应到仓库与零部件

---

## 14. 成本支线当前边界
当前阶段只冻结以下内容：

1. 成本支线同时覆盖保内纯支出和保外有收入
2. 成本支线同时承接报价阶段预估成本与业务执行后的实际成本
3. 当前正式成本项包括服务执行成本、零部件成本、运输成本、平台成本、SIM 卡成本、报废损失、办事处 / 分仓房租
4. 服务经理跨城市处理工单时，工单服务执行成本来源于报销支线中归属于工单的实际费用
5. 当前已知可进入工单服务执行成本的自有人员执行费用包括差旅费、住宿费、打车费
6. 一次出差处理多个工单时，服务执行成本在多个工单之间均摊
7. 驻地城市内无报销费用时，工单服务执行成本记为 0
8. 驻外固定补助不计入单工单服务执行成本，归入间接成本
9. 服务商费用计入工单服务执行成本
10. 服务经理可填写服务商事前报价，但该报价不锁死最终实际成本
11. 外部服务商执行成本只针对已结束工单成立
12. 若服务商最终未执行，则该笔外部服务商执行成本不成立
13. 外部服务商执行成本最终以服务商发票金额入成本
14. 一张服务商发票只对应一张工单
15. 无工单拜访客户类出差形成客户拜访成本，来源于出差申请 / 借款 / 补助 / 报销支线的实际结果
16. 当前已知可进入客户拜访成本的费用包括差旅费、住宿费、打车费
17. 一次出差拜访多个客户时，客户拜访成本在多个客户之间均摊
18. 快递垫付结果进入运输成本
19. 借款本身不直接进入成本，只是报销与补助前置资金安排
20. 消耗品并入零部件成本
21. 零部件成本可对应工单或报价单
22. 运输成本包括零部件运输和客户委托运输两类
23. 平台成本按月上传使用点数，对应机构代码 / 简称
24. SIM 卡成本按月上传账单，对应 ICCID / 卡号
25. 总仓和分仓报废损失按零部件成本处理
26. 间接成本单独管理，当前不展开

当前阶段暂不展开：
- 页面结构
- 字段设计
- 成本报表
- 间接成本明细
- 毛利统计细则
- 成本台账的时间口径
- 与财务系统的对接方式
- 出差申请 / 借款 / 补助 / 报销支线的详细流程
- 服务商报价与发票差异的细化处理规则

## 15. 当前已冻结的关键原则
1. 成本支线既覆盖保内纯支出，也覆盖保外有收入
2. 保内业务可只有成本，没有收入
3. 保外业务需同时看收入、成本和毛利
4. 成本支线既承接报价阶段预估成本，也承接业务执行后的实际成本
5. 需求单发起时需同步带入预估成本信息
6. 服务执行成本分为自有人员执行成本和外部服务商执行成本
7. 自有人员跨城市处理工单时，实际归属于工单的差旅费、住宿费、打车费计入工单服务执行成本
8. 一次出差处理多个工单时，服务执行成本按当前业务规则均摊
9. 驻地城市内无报销费用时，工单服务执行成本记为 0
10. 驻外固定补助不计入单工单服务执行成本，归入间接成本
11. 若工单由外部服务商处理，则服务商费用计入工单服务执行成本
12. 服务经理可在服务商上门前填写服务商事前报价
13. 事前报价属于执行前参考，不锁死最终实际成本
14. 外部服务商执行成本只针对已结束工单成立
15. 若服务商最终未执行，则该笔外部服务商执行成本不成立
16. 一张服务商发票只对应一张工单
17. 发票金额不要求必须与事前报价一致
18. 外部服务商执行成本最终以服务商发票金额入成本
19. 无工单拜访客户类实际费用形成客户拜访成本
20. 客户拜访成本来源于报销支线中归属于客户拜访的实际费用
21. 当前已知可进入客户拜访成本的费用包括差旅费、住宿费、打车费
22. 快递垫付结果进入运输成本
23. 借款本身不直接进入成本，只是报销与补助前置资金安排
24. 零部件成本可对应工单或报价单
25. 运输成本包括零部件运输和客户委托运输
26. 平台成本按月上传使用点数
27. SIM 卡成本按月上传账单
28. 报废损失按零部件成本处理
29. 间接成本单独管理，当前阶段不展开
